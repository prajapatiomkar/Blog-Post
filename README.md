# Project Setup In Your Local Machine
### Steps:
#### 1) Clone the project
    git clone https://github.com/prajapatiomkar/blog-post
#### 2) Change Dir
    cd blog-post
#### 3) Install the dependency
    npm run install 
#### 4) To run project
    npm run dev
#### 5) To build project 
    npm run build
